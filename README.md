# Sitio Web Responsivo Minimalista

Un sitio web moderno, responsivo y minimalista creado con **Bootstrap 5** que incluye múltiples eventos del DOM interactivos.

## 📋 Características

- ✨ **Diseño Minimalista**: Paleta de colores limpia (blanco, negro y grises)
- 📱 **Completamente Responsivo**: Se adapta a cualquier dispositivo (móvil, tablet, desktop)
- 🎨 **Bootstrap 5**: Framework CSS moderno y robusto
- ⚡ **Interactivo**: 15+ eventos del DOM implementados
- 🎯 **Secciones Principales**:
  - Navbar pegajoso
  - Hero Section
  - Servicios
  - Contador interactivo
  - Formulario de contacto
  - Paleta de colores

## 🎯 Eventos del DOM Implementados

1. **Click** - Botón "Explorar" (desplazamiento suave)
2. **Click** - Botones de contador (+, -, Reiniciar)
3. **Submit** - Formulario de contacto con validación
4. **Mouseover/Mouseout** - Hover en tarjetas de servicios
5. **Input** - Detección de cambios en campos de formulario
6. **Click** - Selección de colores con copia al portapapeles
7. **Focus/Blur** - Campos del formulario
8. **Scroll** - Efectos en el navbar
9. **Resize** - Detección de cambios de tamaño de ventana
10. **Load** - Animaciones al cargar la página
11. **Double Click** - Reiniciar contador con doble click
12. **Keypress** - Enviar formulario con Ctrl+Enter
13. **Contextmenu** - Menú contextual personalizado
14. **Dragstart/Dragend** - Elementos arrastrables
15. **Visibilitychange** - Detectar cambios de visibilidad de pestaña

## 🚀 Cómo Usar

1. Abre el archivo `index.html` en tu navegador
2. Explora las diferentes secciones
3. Abre la consola del navegador (F12) para ver los eventos registrados
4. Interactúa con los elementos:
   - Usa el contador
   - Completa el formulario
   - Selecciona colores
   - Redimensiona la ventana
